# Bank-Defaulter-prediction
Bank Defaulter prediction using KFold and SMOTE 


Using previous Bank Customers data we would like to know if the customer will default on the loan or not.
We are using dataset which is a class imbalanced dataset
We will use multiple cross validation techniques like stratified KFold Cross Validation 
And lastly we will use SMOTE and in SMOTE we will use Over sampling.
